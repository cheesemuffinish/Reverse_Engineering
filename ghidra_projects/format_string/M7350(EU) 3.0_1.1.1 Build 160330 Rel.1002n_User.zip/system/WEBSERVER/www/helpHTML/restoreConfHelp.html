Factory Defaults Help

You can restore the device to its factory default settings when necessary.


Click Restore to reset all configuration settings to their default values. All of the settings you have saved will be lost when the default settings are restored.

You can restore the following factory default values:

  • Default username: admin
  • Default password: admin
  • Default IP address: 192.168.0.1
  • Default subnet mask: 255.255.255.0
