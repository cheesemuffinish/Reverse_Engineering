Debug Log - To use the debug log function:

  1. Insert an SD card with at least 100M free space.
  2. Go to the "Advanced > Storage Sharing" page and configure the Wi-Fi access mode.
Note:
  1. Do not enable the debug log function unless required by your technical support.
  2. Do not remove the SD card or change its mode when using the debug log function.
