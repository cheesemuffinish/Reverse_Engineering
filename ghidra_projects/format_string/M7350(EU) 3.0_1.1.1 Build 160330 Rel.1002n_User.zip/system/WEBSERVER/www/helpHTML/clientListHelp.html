Online Clients

The Online Clients page displays the basic information about the Wi-Fi clients that are currently connected to the device.


Client Name - The name of the Wi-Fi client.

MAC Address - The MAC address of the Wi-Fi client.

IP Address - The IP address of the Wi-Fi client.


Click Refresh to update the Wi-Fi client list.

Click Block to add the Wi-Fi client(s) you select to the black list.
