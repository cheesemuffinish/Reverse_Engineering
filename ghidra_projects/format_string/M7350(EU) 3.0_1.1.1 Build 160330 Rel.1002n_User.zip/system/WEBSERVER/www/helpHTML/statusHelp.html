Status Help

Connection Status - Displays the device's connection information, including the connection status, network type (for connected state)/SIM card status (for disconnected state) and IP address.

To change dial-up settings, click Dial-up Settings.

Wi-Fi Status - Displays the device's wireless information, including the SSID, wireless security, wireless band, and current clients.

To change Wi-Fi settings, click Wireless Settings.

Statistics - Displays the device's data usage statistics, including the data monthly used/total used and the data daily used. The data usage accounting may differ from your carrier's, and it is subject to your carrier's statistics.

To change data settings, click Data Settings.

PIN - Personal Identification Number of the SIM card. It consists of 4-8 digits.

PUK - PIN Unlocked Key. It consists of 8 digits.

Remaining Attempts - Shows how many attempts are left for you to try entering the PIN or PUK code. You have 3 attempts at most for entering the PIN code and 10 attempts at most for entering the PUK code.

Auto-unlock PIN - When the PIN code is required upon device restarting, it will be validated automatically once. If validation failed, you need to enter the PIN code on the Status page.
