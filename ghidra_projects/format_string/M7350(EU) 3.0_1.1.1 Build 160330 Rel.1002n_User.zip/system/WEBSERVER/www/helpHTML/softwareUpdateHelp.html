Firmware Update Help

You can update the firmware of the device on this page.


Firmware Version - The current firmware version.

Update via the cloud server - Update your firmware automatically by using the Internet.

Click Check for Updates to search for new firmware version.

New Version - This field displays the version number of the new firmware. You can click the version number to download the firmware or click Release Note to view the update details.

Update via the local server - Update your firmware by using the update file that has been downloaded from website www.tp-link.com.


Click Browse to select the update file on your local computer.

Click Update to update the firmware.
