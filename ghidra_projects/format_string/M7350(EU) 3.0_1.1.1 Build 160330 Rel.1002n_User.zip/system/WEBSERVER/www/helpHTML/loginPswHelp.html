Account Management Help

You can change your login username and password on this page.


Old Username - The old username that is used to log in to the device.

Old Password - The old password that is used to log in to the device.

New Username - The new username you are going to set to log in to the device.

New Password - The new password you are going to set to log in to the device.

Confirm Password - Enter the new password again for confirmation.


Note: If you can’t access the SD card via WiFi after changing the login user name/password, please log out or reboot your computer and then try again.


Click Save to save your settings.
